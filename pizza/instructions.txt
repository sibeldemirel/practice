TP Bootstrap vs Tailwind :

Nous allons travailler en groupe de 3/4 personnes (4 groupes max).
Nous avons une maquette commune à intégrer. Chaque personne du groupe devra intégrer la maquette avec son propre code.
Le groupe pourra s'entraider sur le framework et sa documentation.

Groupe 1 (HTML / CSS) : Robin, Anthony, Romain, Timothé, Jeremy, Mathis, Amandine
Groupe 2 (Bootstrap) : Yasser, Sybel, Fares, Cindy, Cécile
Groupe 3 (Tailwind) : Nawal, Céline
Groupe 4 (Indécis) : Nathanael

Chaque personne a le droit à un joker pour switcher de groupe (de framework). Si vous voulez utiliser Bootstrap et Tailwind en même temps (non conseillé), vous pouvez
configurer Tailwind avec un préfixe sur ce lien : https://tailwindcss.com/docs/configuration#prefix

Pour vous aider, vous pouvez utiliser les salons Discord avec le partage d'écran. Je vous conseille également le plugin Live Share de VS Code.
